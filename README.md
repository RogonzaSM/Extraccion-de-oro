Este proyecto aplica técnicas de Machine Learning en el ámbito de la minería, con el objetivo de desarrollar un modelo capaz de predecir la cantidad de oro que puede extraerse de minerales auríferos. La iniciativa busca optimizar los procesos de extracción mediante el análisis de datos, ofreciendo una herramienta que contribuya a mejorar la eficiencia y la toma de decisiones en la industria minera.
